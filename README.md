# 🍄 Ловец фруктов

Игра на Python с использованием Tkinter, где нужно ловить падающие фрукты и избегать мухоморов.

## 🎮 Управление
- **D** — движение вправо
- **A** — движение влево

## 🎯 Цель игры
- Ловите падающие фрукты — **+1 очко**
- Избегайте мухоморов — **-1 очко**

## 🚀 Запуск
```bash
python catcher.py
📁 Требуемые файлы
